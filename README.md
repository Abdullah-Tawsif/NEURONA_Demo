NEURONA – Creator & Investor Platform

Neurona is a web-based platform that connects creators with investors.
It enables idea sharing, investment opportunities, and platform management through role-based dashboards.

### ✨ Features
- Role-based authentication (Creator, Investor, Admin)
- Idea submission and showcase system
- Investment exploration and interaction
- Admin dashboard for user & idea management
- Responsive design

### 🛠️ Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite

### 📂 Project Structure
  /static        -> CSS, images, assets  
  /templates     -> HTML templates  
  app.py         -> Main Flask app  
  requirements.txt -> Dependencies  

### ⚙️ Installation
  git clone https://github.com/your-username/NEURONA_Demo.git
  cd NEURONA_Demo
  pip install -r requirements.txt
  python app.py
